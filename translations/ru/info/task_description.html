Эта миссия является адаптацией игры "Заполнение" (из коллекции головоломок Саймона Тэтхема). Если вы не знакомы с этой игрой, или просто хотите поиграть, то вам сюда.

У вас есть прямоугольная сетка с цифрами. Вам предстоит заполнить сетку цифрами таким образом, что каждая соединенная область (диагонально расположенные квадраты не считаются соединёнными), образованная из идентичных цифр, имеет площадь, равную этой цифре.

Обратите внимание, что каждая область представлена на сетке по крайней мере одной цифрой.

Пояснения:

  • Заполненная сетка не может содержать нули (поэтому пустые ячейки во входных данных представлены нулями).
  • Единицы всегда одни и уже расставлены на сетке.



[[1, 0, 1, 0, 1, 4, 0, 0],       [[1, 3, 1, 3, 1, 4, 4, 4],
 [3, 0, 2, 3, 0, 0, 0, 4],        [3, 3, 2, 3, 3, 5, 5, 4],
 [0, 0, 0, 1, 0, 0, 5, 1],  ===\  [4, 4, 2, 1, 5, 5, 5, 1],
 [4, 0, 1, 3, 0, 1, 0, 0],  ===/  [4, 4, 1, 3, 3, 1, 8, 8],
 [5, 0, 0, 0, 0, 0, 0, 8],        [5, 5, 5, 2, 3, 8, 8, 8],
 [0, 0, 1, 2, 1, 0, 0, 0]]        [5, 5, 1, 2, 1, 8, 8, 8]]

Входные данные: Список, содержащий списки целых чисел.

Выходные данные: Список/кортеж, содержащий списки/кортежи целых чисел.

Пример: 

checkio([[1, 0, 1, 0, 1, 4, 0, 0],
         [3, 0, 2, 3, 0, 0, 0, 4],
         [0, 0, 0, 1, 0, 0, 5, 1],
         [4, 0, 1, 3, 0, 1, 0, 0],
         [5, 0, 0, 0, 0, 0, 0, 8],
         [0, 0, 1, 2, 1, 0, 0, 0]]) == [[1, 3, 1, 3, 1, 4, 4, 4],
                                        [3, 3, 2, 3, 3, 5, 5, 4],
                                        [4, 4, 2, 1, 5, 5, 5, 1],
                                        [4, 4, 1, 3, 3, 1, 8, 8],
                                        [5, 5, 5, 2, 3, 8, 8, 8],
                                        [5, 5, 1, 2, 1, 8, 8, 8]]

Предусловия:

  • Все предоставленные головоломки имеют решение.
  • 3 ≤ len(grid) ≤ 20 and 3 ≤ len(grid[0]) ≤ 20.
  • all(len(row) == len(grid[0]) for row in grid).
